# AI Job Finder

A smart job search web app built with Python, Flask, and AI.

## Features
- Search jobs from multiple sources (Remotive, Adzuna)
- AI-powered job summaries using Groq
- Smart search with fuzzy matching
- Filter by country, job type, remote
- Beautiful glassmorphism UI

## Tech Stack
- Python + Flask (backend)
- HTML + CSS (frontend)
- Groq AI (job summaries)
- Adzuna API (job listings)
- Remotive API (remote jobs)
